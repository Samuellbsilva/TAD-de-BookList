# 📚 BookList TAD (Java)

Implementação de um **Tipo Abstrato de Dados (TAD)** para gerenciamento de uma lista de livros utilizando **array estático** em Java.

## 🚀 Funcionalidades

- Adicionar livro (ID)
- Remover por ID
- Remover por posição
- Buscar livro
- Atualizar ID
- Ver total de livros
- Verificar se está cheia
- Limpar lista

## 🧠 Conceitos aplicados

- TAD (Tipo Abstrato de Dados)
- Manipulação de arrays
- Controle de capacidade
- Deslocamento manual de elementos
- Interface + implementação
